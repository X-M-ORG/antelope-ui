# Sprite 雪碧图

雪碧图，又叫精灵图，一张图片通过修改定位来显示不同部分。

---

## Props

| name | type   | require | default | desc               |
| ---- | ------ | ------- | ------- | ------------------ |
| size | String | false   |         | 设置图片的显示大小 |
| x    | String | false   |         | x 轴偏移数值       |
| y    | String | false   |         | y 轴偏移数值       |
