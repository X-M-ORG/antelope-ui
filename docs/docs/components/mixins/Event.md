# Event

大多数组件都混合了事件监听。

---

## Event

| name         | desc                                  |
| ------------ | ------------------------------------- |
| a-touchstart | touchstart 事件，移动端触摸开始时触发 |
| a-touchend   | touchend 事件，移动端触摸结束时触发   |
| a-tap        | tap 事件，移动端点击时触发            |

## 额外样式

当 touch 的时候，dom 会新增一个 class： on-touch
